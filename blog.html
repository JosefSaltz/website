From Small Beginnings...

July, 25 2016 9:11PM

Today marks my first day getting involved with Hack Reactor as a student. In an effort to better secure a spot in the Hack Reactor remote program I have started the Hack Reactor Remote Prep course. I've spent the last year learning a lot about the web development world but unfortunately it wasn't enough to pass the technical interview. I'm hoping that Hack Reactor Prep will be enough to patch the holes I've made from a self-taught curriculum and let me truly find out if I have what it takes to become a real software engineer. I've spent last Saturday setting up my Linux partition for development work in Ubunut and beginng the pre-course work which is helping me warm up my HTML/CSS skill set in the form of slowly putting together this portfolio/self-bio web page. At some point hopefully in the near future this page will look like a real fleshed out web developer's site instead of the hollow html skeleton it looks like now. I am greatly looking forward to learning to pump out full fledged web content and applications with a team of other developers these coming weeks. In lieu with getting my Linux installation reinstalled this weekend I'm also posting the awesome Linus Torvalds TED talk!
